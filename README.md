# Customer Segmentation Analysis

## Overview
This project performs customer segmentation using K-Means clustering based on customer income and spending behavior.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Features
- Data preprocessing
- Feature scaling using StandardScaler
- Elbow method for optimal cluster selection
- Customer segmentation using K-Means clustering
- Business insights and visualization

## Dataset
Mall Customers Dataset

## Business Insights
- Identified premium customers
- Identified low-engagement customers
- Analyzed customer spending behavior
- Generated customer segmentation visualizations
